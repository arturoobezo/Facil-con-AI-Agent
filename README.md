Facil con AI Agent.

A mini-agent that uses Ollama in the background to run.
You can use all the models available in Ollama with the advantage of having a sandbox where you can preview your project.

To install it, just run the file: Facil_con_AI_Agent_Run.bat

The Ollama download will begin automatically.

If you get any errors, manually download and install Ollama, run the mini-agent again, and you're done.

If you want to learn how to use this mini-agent, you can watch the video tutorial:
